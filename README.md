## 🚀 Overview

**Otter Notes** is a time management solution specifically designed to help students stay organized. With no installation required, this app allows you to manage deadlines with a smart notification system directly through your browser.

### 🎯 Project Goals
1. **Intuitive:** Developing an interface that is easy to understand with zero learning curve.
2. **Organized:** Helping users manage academic priorities more effectively.
3. **Disciplined:** Reducing the risk of late submissions through an automated reminder system.

## ✨ Key Features

- 📝 **Smart Input** — Easily manage titles, descriptions, dates, and deadline times.
- 🗂️ **Card-Based List** — A clean and modern task list display using a card layout.
- 🔔 **Smart Notification** — Automatic reminders at **D-3** and **D-1** before the deadline.
- 🌓 **Dark Mode Support** — Comfortable for late-night study sessions.
- 🧹 **One-Click Manage** — Instant task completion check and deletion.

- 🎓 **University Students** — Manage complex course assignments and organizations.
- 🏫 **Students** — Track daily homework and exam schedules.
- 👤 **General Users** — Anyone who handles multiple deadlines in their activities.
